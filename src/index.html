Unlocking the Potential of Face Recognition in OpenCV

This research project was focused on evaluating and proposing the optimal face recognition algorithm using the OpenCV library. The project compared conventional and deep learning models, such as Eigenfaces, Fisherfaces, LBPH, MTCNN, and VGG16. Additionally, the project analyzed precision in identifying facial landmarks using machine learning models like Haar Cascades and deep learning libraries like TensorFlow and Keras.

Additionally, the project featured a unique aspect where the user's face could be matched with a dataset of celebrity faces, and the results were compared using various algorithms to determine the most accurate one.

This project was published in ICBDM 2023 (International Conference on Big Data Management), held in Bangkok, Thailand.

This project was also rewarded as the Best Innovative Research Award at the 8th CS Demo Day conducted by the School of Computer Science UWindsor.

Smart Device Management

The "Smart Device Management" mobile app is an innovative solution developed to monitor registered smart devices with real-time usage statistics and malfunction alerts. With this app, users can have a one-stop-shop for all their smart home needs. The app offers a convenient way to check the status of smart devices, including smart TVs, refrigerators, and more, all in one place. Additionally, the app provides reporting for up to 30 days, allowing users to track their smart device activity through a comprehensive report.

Real Estate Web Search Engine

The Real Estate Web Search Engine project is a highly advanced and optimized search engine that helps users search for properties based on specific criteria. Utilizing various Java libraries, including Hash maps, Arraylist, JSoup library, Edit Distance algorithm, and Stream API, the project extracted data, performed spell check, and sorted search results. Users can input their desired location, number of bedrooms, and price range to get a comprehensive listing of matching properties. The Edit Distance algorithm implemented in the project helps achieve high accuracy levels by correcting spelling mistakes and typos in search queries.
